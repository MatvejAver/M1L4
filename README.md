# Бот покеионов!
##Описание
###Для чего создан?
Этот бот создан для того,**чтобы воевать с другими покемонами**, и _просто веселиться_ :)

##Функционал
В этом боте есть несколько функций:
1. ***/go*** она создана для ~скучания~ создания своего **Покемона!**
> (создавать несколько покемонов одному пользователю нельзя!)
2. ***/attack*** - для *АТАКИ* других **ПОКЕМОНОВ**!
> Эту команду надо отправлять в ответ на сообщение того, кого вы хотите атаковать!
3. ***/info*** - выводит информацию о **вашем покемоне**!
> Тут ничего нету! <sup>:) я тут</sup>
4. ***/feed*** - команда для *кормления* своего **покемона**!
> (кормление восстанавливает hp)
